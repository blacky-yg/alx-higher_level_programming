===========================
Test your MyList class
===========================

=====
Description
=====

``MyList`` is a set of list and the function ``print_sorted`` prints the sorted list

=====
Usage
=====

First, you need to import the ``MyList`` from the module

    >>> MyList = __import__("1-my_list").MyList

Now, you can use it like following:

    >>> a = MyList()
    >>> a
    []

    >>> a.append(1)
    >>> a
    [1]

    >>> a.append(4)
    >>> a.append(3)
    >>> a
    [1, 4, 3]

    >>> a.print_sorted()
    [1, 3, 4]

    >>> a.append("b")
    >>> a
    [1, 3, 4, 'b']

    >>> a.print_sorted()
    Traceback (most recent call last):
        ...
    TypeError: unorderable types: str() < int()

    >>> a = MyList()
    >>> a.append(4)
    >>> a.append(3)
    >>> a.append(-2)
    >>> a.append(0)
    >>> a
    [4, 3, -2, 0]

    >>> a.print_sorted()
    [-2, 0, 3, 4]

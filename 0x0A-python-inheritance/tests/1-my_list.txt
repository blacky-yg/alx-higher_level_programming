===========================
Test your MyList class
===========================

=====
Description
=====

``MyList`` is a set of list and the function ``print_sorted`` prints the sorted list

=====
Usage
=====

First, you need to import the ``MyList`` from the module

    >>> MyList = __import__("1-my_list").MyList

Now, you can use it like following:

    >>> a = MyList()
    >>> a
    []

    >>> a.append(1)
    >>> a
    [1]

    >>> a.append(4)
    >>> a.append(3)
    >>> a
    [1, 4, 3]

    >>> a.print_sorted()
    [1, 3, 4]

    >>> a.append("b")
    >>> a
    [1, 3, 4, 'b']

    >>> a.print_sorted()
    Traceback (most recent call last):
        ...
    TypeError: unorderable types: str() < int()

    >>> b = MyList()
    >>> b.append(4)
    >>> b.append(3)
    >>> b.append(-2)
    >>> b.append(0)
    >>> b
    [4, 3, -2, 0]

    >>> b.print_sorted()
    [-2, 0, 3, 4]

    >>> l = MyList()
    >>> l.append(4)
    >>> l.append(3)
    >>> l.append(2)
    >>> l.append(1)
    >>> print(l)
    [4, 3, 2, 1]
    >>> l.print_sorted()
    [1, 2, 3, 4]
    >>> print(l)
    [4, 3, 2, 1]

    >>> l.print_sorted(1)
    Traceback (most recent call last):
    ...
    TypeError: print_sorted() takes 1 positional argument but 2 were given
